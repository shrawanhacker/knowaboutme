## 📥 How to Publish on GitHub Pages
1. Create a new public repository on GitHub named `portfolio` or `<your-username>.github.io`.
2. Push `index.html`, `style.css`, `script.js`, `README.md`, and your `resume.pdf` to the repository.
3. Go to **Settings** -> **Pages**.
4. Set the branch to `main` (or `master`) and click **Save**.
5. Your portfolio will be live in 1-2 minutes!
